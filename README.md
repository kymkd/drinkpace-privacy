## DrinkPace について

DrinkPace は、飲酒習慣を「整える」ための管理アプリです。飲んだものを手軽に記録し、休肝日スケジュールの達成状況を可視化することで、無理なく健康的な飲酒ペースを見つけることをサポートします。
